# PAUMarket QR Landing Page

Bu klasör, posterdeki QR kodun yönleneceği küçük tanıtım ve iletişim sayfasını içerir.

## Yayina alma

GitHub üzerinde repo ayarlarından GitHub Pages kaynağını `main` branch ve
`/docs` klasörü olarak seçince sayfa şu adresten açılır:

```text
https://hasantogmus.github.io/pau-market/qr/
```

## Linkleri guncelleme

GitHub veya iletişim bilgileri değişirse `config.js` içindeki alanları güncelle.

Poster QR kodu için önerilen hedef link:

```text
https://hasantogmus.github.io/pau-market/qr/
```

Hazır QR görseli:

```text
docs/qr/paumarket-qr.png
```
